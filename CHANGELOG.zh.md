## 📦 v1.1.2

-   您可以安装 `1.1.2-alpha.0` 版本来体验
-   添加选项：

    -   `-t`, `o/a` 获取仓库类型，默认值 `a` `(o: owner，a: all)`

-   如何指定获取仓库类型：

```sh
# 获取个人所有仓库
del-repos -t o

# 获取所有仓库（包含合作者和组织成员仓库）
del-repos
# 或
del-repos -t a
```
